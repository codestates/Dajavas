# 🐠 소개

## 초보자를 위한 낚시 웹 사이트 🪝

- 잡은 물고기를 기록하고 랭킹으로 순위를 확인 할 수 있습니다.
- 자주 가는 낚시 포인트를 저장할 수 있습니다.
- 위치기반 서비스로 근처의 낚시터를 확인 할 수 있습니다.
- 포인트 클릭시 해당 지역의 날씨와 풍향등을 확인 할 수 있습니다.
- 물고기들의 각종 정보와 금어기를 확인 할 수 있습니다.
- 초보 낚시꾼들을 위한 아이템 체크리스트를 제공합니다.
